NUGU Albert AI service
- ALRIMA (알리마)

구글 캘린더에 등록된 일정 알림 서비스
Alarm service for schedules registered on google calendar



기능
1. 특정 날짜의 전체 일정 검색
2. 특정 날짜의 업무 일정 검색
3. 특정 날짜의 개인 일정 검색
4. 특정 날짜의 소지품 검색
5. 특정 일정이 등록된 날짜 검색



실행 방법
1. putty 실행
2. main.py를 실행하여 구글 캘린더 로그인 토큰 저장
3. reminder1.py를 실행
4. 누구 플레이에서 '알리마' 서비스 플레이
